Scatter Plot

What is a Scatter Plot?
A scatter plot is a graph that uses dots to represent individual data points on a two-dimensional plane.
Each dot represents a data point, and the position of the dot is determined by two variables, one for the x-axis and one for the y-axis.
Scatter plots are useful for showing correlations or relationships between two variables.

Key Characteristics of Scatter Plots:
• Each data point is represented by a dot.
• The position of the dot corresponds to values on the x and y axes.
• Scatter plots are used to show relationships or correlations between two variables.

Components of a Scatter Plot:
• Vertical Axis (Y-Axis) – Represents one numerical variable.
• Horizontal Axis (X-Axis) – Represents another numerical variable.
• Dots – Each dot represents a data point.
• Title – Provides insight into the purpose of the plot.

Types of Scatter Plots:
1. Simple Scatter Plot:
A basic scatter plot showing data points.
2. Bubble Plot:
A variation of the scatter plot where each data point is represented by a bubble.

Properties of Scatter Plots:
• Each data point is represented by a dot.
• The position of the dots is determined by the values of the two variables.
• Scatter plots can show trends, correlations, and clusters in data.

Uses of Scatter Plots:
Scatter plots are used to identify relationships between two variables, detect trends, and highlight clusters or outliers.
They are effective for visualizing correlations, regression analysis, and distribution.

Real-Life Applications:
• Analyzing the correlation between hours of study and exam scores.
• Showing the relationship between advertising budget and sales.
• Displaying the distribution of ages in a population.

Advantages & Disadvantages of Scatter Plots:
Advantages:
• Great for showing correlations and relationships.
• Helps identify trends, clusters, and outliers.
• Simple and effective for comparing two variables.
Disadvantages:
• Can become cluttered with too many data points.
• Not suitable for categorical data.

Drawing a Scatter Plot – Step by Step:
Example data: X: 1, 2, 3; Y: 4, 5, 6.

1. Choose a Title – e.g., "Height vs Weight."

2. Draw Axes – Horizontal (X-axis) and Vertical (Y-axis).

3. Label Axes – X-axis: Height; Y-axis: Weight.

4. Plot Data Points – Mark each data point on the graph.

5. Analyze Relationships – Look for trends, clusters, or outliers.
