Line Chart

What is a Line Chart?
A line chart, also known as a line graph, is a visual representation of data using points connected by straight lines.
This type of chart is useful for showing trends over time or continuous data.
Line charts display the relationship between two variables, where one is typically the time on the x-axis, and the other variable is represented on the y-axis.

Key Characteristics of Line Charts:
• They show data points connected by a line to display trends.
• They are especially effective for visualizing continuous data over time.
• The x-axis usually represents time, and the y-axis represents numerical values.

Components of a Line Chart:
• Vertical Axis (Y-Axis) – Represents numerical values.
• Horizontal Axis (X-Axis) – Represents time or other continuous data.
• Title – Provides insight into the purpose of the chart.
• Data Points – Each point represents a data value.
• Line – Connects data points to show the trend.

Types of Line Charts:
1. Simple Line Chart:
A single line is drawn to represent one data series.
2. Multiple Line Chart:
Multiple lines are drawn to compare different data series.
3. Stacked Line Chart:
Displays multiple data series stacked on top of each other.
4. Dot Plot:
Points are plotted with markers, not connected by a line.

Properties of Line Charts:
• Data points are connected by straight lines.
• The x-axis typically represents time or continuous intervals.
• The y-axis represents values corresponding to each data point.

Uses of Line Charts:
Line charts are commonly used to track changes over time, compare multiple variables, or illustrate trends.
They are particularly useful for data that has a clear sequence, like stock prices, weather patterns, or sales data.

Real-Life Applications:
• Tracking stock prices over time.
• Showing changes in temperature over a day or year.
• Displaying sales trends for a business.

Advantages & Disadvantages of Line Charts:
Advantages:
• Clearly displays trends and patterns over time.
• Makes it easy to compare multiple data sets.
• Ideal for continuous data.
Disadvantages:
• Can be difficult to interpret with too many lines.
• Less effective for comparing discrete data points.

Drawing a Line Chart – Step by Step:
Example data: January: 10, February: 15, March: 30, April: 50.

1. Choose a Title – e.g., "Number of Sales Over Time."

2. Draw Axes – Horizontal (X-axis) and Vertical (Y-axis).

3. Label Axes – X-axis: Months; Y-axis: Sales numbers.

4. Set a Scale – Choose appropriate intervals for the vertical axis (e.g., 10, 20, 30, etc.).

5. Plot Data Points – Mark each month's sales on the chart.

6. Draw a Line – Connect the points to show the trend.
